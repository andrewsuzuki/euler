# euler

[Project Euler](https://projecteuler.net/) solutions by Andrew Suzuki, written in Python, Ruby, Javascript, Clojure, Java, and C.

## Testing

Run

```
$ cd test
$ ./test_all
```

to test all solutions in every language found in solutions/.
